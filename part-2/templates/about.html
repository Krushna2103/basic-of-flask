The render_template() Function

Here's how we render this page in Python:

@app.route('/about')
def about():
    return render_template('about.html')

How Templates Work

Flask uses a template engine called Jinja2. Right now we're just rendering static HTML, but in the next parts, we'll learn how to:

  • Pass variables from Python to HTML
  • Use loops to display lists
  • Use conditionals to show/hide content
  • Create reusable template layouts
